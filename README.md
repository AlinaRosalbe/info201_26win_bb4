# info201_26win_bb4

This project was completed as the final project for INFO 201. It investigates Alcohol Consumption and Alzheimer’s Trends by analyzing state-level data across the United States. Through data wrangling and visualization, the report highlights regional patterns in alcohol use, changes in Alzheimer’s-related indicators, and the possible relationship between these two variables.
